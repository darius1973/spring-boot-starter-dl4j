DL4J Spring Boot Starter

A Spring Boot starter that enables declarative neural network configuration via YAML, powered by Deeplearning4j.

Status: Beta (v0.1.0)

🚀 Quick Example


dl4j:
  input-size: 3
  model:
    layers:
      - type: dense
        size: 16
      - type: output
        size: 1
        activation: sigmoid
        loss: mse

  • API Documentation (Javadoc)

✨ Features

  • YAML-driven neural network definition
  • Spring Boot auto-configuration
  • Training and prediction API
  • Model persistence (save/load)

🧠 About

This project aims to make Deeplearning4j easier to use in modern Spring Boot applications, without requiring extensive boilerplate code.
