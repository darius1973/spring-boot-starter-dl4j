# Spring Boot Starter DL4J

A custom Spring Boot starter for integrating Deep Learning capabilities using Deeplearning4j.

This starter simplifies the setup of neural networks, training pipelines, and inference services inside Spring Boot applications.

---

## 🚀 Features

- Auto-configuration for Deeplearning4j
- Pre-configured ND4J backend (CPU support)
- Easy model training service
- REST-ready inference support
- Spring Boot Actuator integration
- Extensible architecture for custom models

---

## 🧠 Tech Stack

- Spring Boot
- Deeplearning4j (DL4J)
- ND4J (Numerical computing backend)
- Java 17+

---

## 📦 Installation

Add the dependency to your Spring Boot project:

```xml
<dependency>
    <groupId>io.github.darius1973</groupId>
    <artifactId>dl4j-spring-boot-starter</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
